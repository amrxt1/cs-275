# 🌐 CCHive - Student Community Platform

[![React](https://img.shields.io/badge/Frontend-React-blue.svg)](https://react.dev/)
[![Ruby on Rails](https://img.shields.io/badge/Backend-Ruby%20on%20Rails-red.svg)](https://rubyonrails.org/)
[![License](https://img.shields.io/badge/License-MIT-green.svg)](LICENSE)

CCHive is a student community website designed to connect learners for **buying/selling study materials, forming study groups, seeking peer tutoring, and providing course/instructor reviews**. Built with **React** for the frontend and **Ruby on Rails** for the backend, this platform aims to streamline student interactions on campus.

## 📂 Project Structure

```
CCHive/
│── cchive_frontend/      # React application (Frontend)
│── cchive_backend/       # Ruby on Rails API (Backend)
│── docs/          # Documentation and UML diagrams
│── .gitignore     # Git ignore file
│── README.md      # Project overview
│── LICENSE        # License file
```

---

## 🚀 Features

✔ **User Registration & Authentication** (Email verification, secure login)  
✔ **Marketplace for Books** (Buy/sell textbooks, filters & search)  
✔ **Study Group Platform** (Create, join, and manage study groups)  
✔ **Peer Tutoring System** (Volunteer tutors, appointment scheduling)  
✔ **Course & Instructor Reviews** (Rate and review professors and courses)  
✔ **Real-time Chat & Notifications** (Messaging between users)

## 🎯 Installation & Setup

### 🔹 Backend (Rails API)
```sh
cd backend
bundle install
rails db:create db:migrate db:seed
rails server
```

> Ensure you have **Ruby 3.x** and **Rails 7.x** installed.

### 🔹 Frontend (React)
```sh
cd frontend
npm install
npm start
```

> Ensure you have **Node.js 16+** installed.
